# JUMPSHOT

![Game Screenshot](Screenshots/screenshot1.png)  <!-- optional image -->

## Overview
**JUMPSHOT** is a movement based FPS shooter where you shoot down targets with various weapons while trying to stay in the air. 
Some targets are big, and others are small. Some explode, and others shoot at you. You rack up more points by staying in the air. You can use these points to buy weapons.

## Features
-Tight movement physics
-Hitscan weapons
-Various utility/grenades and abilities
-Wallrunning

## Project Structure
- **Assets/** – All scripts, prefabs, scenes, textures, and models  
- **Packages/** – Unity package dependencies  
- **ProjectSettings/** – Unity project configuration  
- **.gitignore** – Unity-specific ignores (Library, Temp, Obj, Build, etc.)

## Requirements
- **Unity Version:** 2023.3.33f1 or later
- **Platform:** Windows

## Getting Started
You can download the project or download the game itself on GameJolt: https://gamejolt.com/games/jumpshot/1012107

GAMEJOLT INSTRUCTIONS:
1. Download the Zip file and unzip it to a new folder on your desktop.
2. Open the "Real" Folder
3. Open the JUMPSHOT!.exe file
4. If Windows defender opens, just run the file anyway. I promise its not a virus. Im not smart enough to make those anyway.
5. Play the game.
6. In order to exit, either quit from the main menu, or press ESC while playing to open the menu, then navigate to Other > Quit

UNITY PROJECT INSTRUCTIONS:
1. Clone the repository using Git (Command Below)
2. Open Unity Hub.
3. Click Add project → Open project from disk.
4. Navigate to the folder you just cloned or extracted.
5. Unity will load the project and import all necessary assets and packages.
